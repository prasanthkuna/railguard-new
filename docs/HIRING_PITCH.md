# Railguard — Hiring Pitch

Railguard is an ERC-7579/ERC-4337 safety module for AI-agent stablecoin payments.

**What it proves:**
- On-chain session spending limits with nonce-lane isolation
- OPA/Rego policy + deterministic audit receipts
- Adversarial Foundry tests for batch injection, replay, and spend drift
- Dual-layer model: off-chain policy advises, on-chain hook enforces

**Demo:** One allowed USDC payment + blocked attacks (batch injection, wrong recipient, cumulative cap).
